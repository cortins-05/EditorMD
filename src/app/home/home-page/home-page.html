This is a real-time Markdown editor I built using Angular, designed to provide a smooth and intuitive writing experience. It allows me and other users to create, edit, preview, and export Markdown content effortlessly. In this project, I implemented:
  • Live Markdown Preview:
    As I type in the text area, the editor instantly renders the formatted output in a preview pane.
  • File Management:
    I can open, save, and save as Markdown files directly from the application.
  • PDF Export:
    I added functionality to export documents by converting Markdown (through HTML rendering) into PDF files.
  • Modern UI:
    I designed a responsive, minimalistic interface that makes writing and editing simple and enjoyable.
  • Utility Services:
    I created Angular services like SaveAs and WriteUtils to handle file operations and exporting logic in an organized way.
